This is a person authentication script with validations for the following:
- It detects if the user is disabled/ inactive in the LDAP/ AD and gives a message that your account is disabled.
- Each time pops up an option to update the user password. During password update it checks for the password complexity and gives a message to the user if the password is not upto the expected complexity level. If a user input a complex password then it updates the password.
